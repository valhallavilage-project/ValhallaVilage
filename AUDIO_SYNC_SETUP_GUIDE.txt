================================================================================
ЧТО НАСТРАИВАТЬ
================================================================================

В Inspector у MainCharacterAudioView есть звуки:

1. Walk Sound (звук шагов)
   - Is Loop: ✅ (должна быть галочка)
   - Animation Cycle Duration: [НАСТРОИТЬ!]

2. Gather Sound (сбор ресурсов руками)
   - Is Loop: ✅
   - Animation Cycle Duration: [НАСТРОИТЬ!]

3. Axe Sound (рубка топором)
   - Is Loop: ✅
   - Animation Cycle Duration: [НАСТРОИТЬ!]

4. Pick Axe Sound (кирка по камню)
   - Is Loop: ✅
   - Animation Cycle Duration: [НАСТРОИТЬ!]

5. Attack Sound (атака мечом)
   - Is Loop: ❌ (без галочки - не нужно настраивать)

6. Death Sound (смерть)
   - Is Loop: ❌ (не нужно настраивать)

================================================================================
КАК УЗНАТЬ ДЛИТЕЛЬНОСТЬ ЦИКЛА АНИМАЦИИ
================================================================================

ВАРИАНТ 1: Через Animation Controller

1. Откройте префаб персонажа (скин)
2. Найдите компонент Animator
3. Кликните на Controller (например "MC_Male_Animator")
4. Откроется Animation Controller окно
5. Найдите нужную анимацию (Walk, Chop, Pickaxe, Gather)
6. Кликните на неё
7. В Inspector справа увидите:
   - Motion: [название анимации]
   - Speed: 1.0 (множитель скорости)
   - Длительность: X секунд

Например:
   Walk: 0.5 секунды
   Chop: 1.0 секунда
   Pickaxe: 0.8 секунд

ВАРИАНТ 2: Через Animation окно

1. Window → Animation → Animation
2. Выберите объект с Animator
3. В Animation окне выберите нужную анимацию
4. Посмотрите длительность на временной шкале

ВАРИАНТ 3: Подобрать на слух (проще всего!)

1. Запустите игру (Play)
2. Начните рубить дерево/камень
3. Посмотрите сколько раз персонаж замахивается за 1 секунду
4. Если 2 раза за секунду → длительность = 0.5
5. Если 1 раз за секунду → длительность = 1.0

================================================================================
ПРИМЕРЫ НАСТРОЙКИ (ТИПИЧНЫЕ ЗНАЧЕНИЯ)
================================================================================

Walk Sound:
   Animation Cycle Duration: 0.5
   (шаги обычно быстрые - 2 шага в секунду)

Axe Sound (рубка топором):
   Animation Cycle Duration: 1.0
   (замах и удар - обычно ~1 секунда)

Pick Axe Sound (кирка):
   Animation Cycle Duration: 0.8
   (чуть быстрее топора)

Gather Sound (сбор руками):
   Animation Cycle Duration: 0.6
   (быстрее чем инструменты)

================================================================================
ЧТО ПРОИЗОЙДЁТ ПОСЛЕ НАСТРОЙКИ
================================================================================

БЫЛО (без Animation Cycle Duration):
- Звук Loop зацикливается непрерывно
- НЕТ синхронизации с анимацией
- Звук может играть когда персонаж ещё замахивается
- Или звук заканчивается раньше удара

СТАЛО (с Animation Cycle Duration):
- Звук проигрывается КАЖДЫЕ X секунд (как тик таймера)
- СИНХРОННО с циклом анимации
- Звук удара точно когда персонаж ударяет
- Звук шагов точно когда нога касается земли

================================================================================
ПРИМЕР: ЗВУК ТОПОРА
================================================================================

Настройки:
   Axe Sound:
     - Clip: [sound_axe_hit.wav]
     - Is Loop: ✅
     - Animation Cycle Duration: 1.0

Что происходит в игре:
   0.0 сек: Персонаж начинает рубить → звук "тук!"
   1.0 сек: Цикл анимации повторился → снова "тук!"
   2.0 сек: Ещё цикл → "тук!"
   ...
   Персонаж перестал рубить → звуки останавливаются

ИДЕАЛЬНАЯ СИНХРОНИЗАЦИЯ!

================================================================================
ТЕСТИРОВАНИЕ
================================================================================

После настройки всех звуков:

1. Запустите игру (Play)
2. Походите → звук шагов в ритм ногам?
3. Рубите дерево → звук топора в момент удара?
4. Бейте камень киркой → звук синхронен?
5. Соберите траву руками → звук в такт?

Если НЕ синхронизировано:
- Измените Animation Cycle Duration (увеличьте/уменьшите на 0.1-0.2)
- Протестируйте снова
- Подбирайте пока не будет идеально

================================================================================
ВАЖНО
================================================================================

⚠️ Это поле появится ТОЛЬКО У ЗВУКОВ С ГАЛОЧКОЙ "Is Loop"!

Для звуков без Loop (Attack, Death) - это поле не нужно и не появится.

1. Открыть префаб SimpleMovementController
2. Найти компонент MainCharacterAudioView
3. Для каждого зацикленного звука (Walk, Axe, Pickaxe, Gather):

   - Подстрой значение пока не будет синхронно

Примерно займёт 10-15 минут настройки всех звуков.
