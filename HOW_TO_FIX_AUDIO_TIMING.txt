================================================================================
КАК ИСПРАВИТЬ: ЗВУК РУБКИ ОПЕРЕЖАЕТ АНИМАЦИЮ
================================================================================

ПРОБЛЕМА:
Звук топора играет слишком часто, опережает анимацию удара.

ПРИЧИНА:
Animation Cycle Duration настроен слишком маленький (звук тикает чаще чем анимация).

================================================================================
РЕШЕНИЕ: УВЕЛИЧИТЬ ANIMATION CYCLE DURATION
================================================================================

ГДЕ НАСТРАИВАТЬ:

1. Project → Assets → PROJECTS → L2Farm → Prefabs → MainCharacter
2. Двойной клик на "SimpleMovementController.prefab"
3. В Hierarchy найдите объект с компонентом "MainCharacterAudioView"
4. Кликните на него
5. В Inspector (справа) найдите:

   Axe Sound (звук топора)
     - Clip: [звук топора]
     - Is Loop: ✅
     - Animation Cycle Duration: [ТЕКУЩЕЕ ЗНАЧЕНИЕ]

================================================================================
КАК ПОДОБРАТЬ ПРАВИЛЬНОЕ ЗНАЧЕНИЕ
================================================================================

МЕТОД 1: На слух (САМЫЙ ПРОСТОЙ)

Текущее значение (например): 0.8

1. Запустите игру (Play)
2. Начните рубить дерево
3. Слушайте: звук опережает удар?

ЕСЛИ ОПЕРЕЖАЕТ (звук раньше анимации):
   → УВЕЛИЧЬТЕ значение
   → Было 0.8 → попробуйте 1.0
   → Было 1.0 → попробуйте 1.2

ЕСЛИ ОТСТАЁТ (звук после анимации):
   → УМЕНЬШИТЕ значение
   → Было 1.2 → попробуйте 1.0
   → Было 1.0 → попробуйте 0.8

4. Сохраните (Ctrl+S)
5. Протестируйте снова
6. Повторяйте пока не будет идеально синхронно

МЕТОД 2: Точное измерение

1. Window → Animation → Animation
2. Выберите префаб скина персонажа с Animator
3. В Animation окне выберите анимацию "Chop" (рубка)
4. Посмотрите на временной шкале внизу - сколько секунд от начала до конца
5. Это и есть Animation Cycle Duration

================================================================================
РЕКОМЕНДУЕМЫЕ ЗНАЧЕНИЯ (НАЧАЛЬНЫЕ)
================================================================================

Walk Sound (шаги):
   Animation Cycle Duration: 0.5
   (обычно 2 шага в секунду)

Axe Sound (топор):
   Animation Cycle Duration: 1.2
   ← ПОПРОБУЙТЕ ЭТО! (было 1.0, увеличили)

Pick Axe Sound (кирка):
   Animation Cycle Duration: 1.0

Gather Sound (руками):
   Animation Cycle Duration: 0.8

================================================================================
ПРИМЕР НАСТРОЙКИ ТОПОРА
================================================================================

ШАГ 1: Текущее значение
   Axe Sound → Animation Cycle Duration: 1.0

ШАГ 2: Тестируем
   Play → рубим дерево → звук опережает на ~0.2 сек

ШАГ 3: Увеличиваем
   Animation Cycle Duration: 1.2 (было 1.0)

ШАГ 4: Тестируем снова
   Play → рубим → ещё немного опережает

ШАГ 5: Ещё увеличиваем
   Animation Cycle Duration: 1.4

ШАГ 6: Тестируем
   Play → рубим → ИДЕАЛЬНО СИНХРОННО! ✅

ШАГ 7: Сохраняем (Ctrl+S)
   Готово!

================================================================================
ВАЖНЫЕ СОВЕТЫ
================================================================================

1. Меняйте по 0.2 секунды за раз (0.8 → 1.0 → 1.2)
   Не прыгайте сразу на большие значения

2. Тестируйте после КАЖДОГО изменения
   Не меняйте сразу все звуки - по одному!

3. Если совсем не понятно - начните с 1.5 для рубки
   Потом уменьшайте пока не станет синхронно

4. Сохраняйте (Ctrl+S) после каждого изменения!

================================================================================
ПРОВЕРКА ЧТО СИСТЕМА РАБОТАЕТ
